# Abundance 项目

## 项目简介
“Abundance”是一个融合多种优秀框架特性的项目，基于 PyQt 框架进行构建，旨在提供高效、灵活且易于维护的开发解决方案。它借鉴了 Vue3 的目录结构和规范，使得代码组织更加清晰有序；同时参考 Spring Boot 的多模块设计理念，实现了良好的模块解耦，便于功能的扩展与维护。

## 主要特性
- **启动配置加载**：启动时自动从配置文件加载参数，就像 Spring Boot 一样，为项目运行提供灵活的配置支持。
- **Banner 打印**：启动时展示定制的 banner，增强项目启动的辨识度和仪式感。
- **依赖管理**：使用 Poetry 作为包管理器，高效管理项目依赖，确保项目环境的一致性和稳定性。

## 目录结构
- **bin**：存放可执行文件。
- **common**：通用代码模块，存放项目中可复用的代码。
- **cxframework**：框架核心代码。
- **doc**：项目文档目录，包含使用说明、设计文档等。
- **pyqt**：
  - **public**：存放公共资源。
  - **src**：
    - **api**：处理接口相关逻辑。
    - **assets**：存放资源文件，如图像、样式表等。
    - **components**：应用组件目录。
    - **directive**：指令相关代码。
    - **layout**：布局相关代码。
    - **plugins**：插件目录。
    - **router**：路由相关代码。
    - **store**：数据存储相关代码。
    - **utils**：工具函数目录。
    - **views**：视图相关代码。
- **resources**：
  - **config**：存放配置文件。
  - **banner.txt**：可能是启动 banner 的相关文本。
- **main.py**：项目的主启动文件。
- **poetry.lock**：Poetry 生成的依赖锁定文件，确保依赖版本的一致性。
- **pyproject.toml**：Poetry 的项目配置文件，定义项目元数据和依赖关系。

## 使用说明
1. **安装依赖**：在项目根目录下，运行 `poetry install` 命令安装项目所需的依赖。
2. **配置文件**：根据项目需求，在 `resources/config` 目录下配置相关参数。
3. **启动项目**：运行 `python main.py` 启动项目。
